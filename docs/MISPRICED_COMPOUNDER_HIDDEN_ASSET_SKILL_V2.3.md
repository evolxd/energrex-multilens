# MISPRICED COMPOUNDER & HIDDEN ASSET SELECTOR V2.3

## 定位

本 Skill 是 Energrex 的 **误价发现与特殊机会引擎**，不是现有五维评分的第六维，也不产生第二个最终投资总分。它负责发现市场可能看错的地方、执行硬门控、形成可证伪假设，并把通过的标的交给现有五维、估值、IDI 与组合模块。

## 六类机会路径（只路由，不加分）

1. 伟大公司短期困境
2. 现金流回购时间机器
3. 隐藏资产或稀缺权利
4. 资本配置重估
5. 市场隐含预期错位
6. 清算价值与资产套利

每家公司只指定一条主路径，其他仅作为辅助路径，不因符合多条路径重复加分。

## 五大支柱与硬门控

| 支柱 | 权重 | 核心问题 |
|---|---:|---|
| P1 生意真相 | 25 | 生意如何赚钱，谁使用、谁付款、现金是否真实 |
| P2 生存能力 | 20 | 公司能否活到价值兑现，债务与流动性是否安全 |
| P3 误价认知 | 20 | 市场具体错在哪个可验证变量 |
| P4 价值捕获 | 20 | 价值最终是否归普通股，如何兑现 |
| P5 价格赔率 | 15 | 当前价格的安全边际、赔率和等待成本 |

任一硬门 Gate=FAIL 时，不计算研究优先级分，也不能由其他高分覆盖。该分仅用于研究排序，最终投资结论仍由 Energrex IDI 输出。

## 付款方经济学审计（横向机制，不独立加分）

必须回答：

- 使用者是谁？
- 经济付款方是谁？
- 谁决定价格？
- 支付是否刚性？
- 付款方信用和集中度如何？
- 政策依赖度多高？
- 成本上涨能否转嫁？
- 回款周期和坏账风险如何？
- 付款结构能支持多大合理杠杆？

注意：政府或保险付款并不自动等于定价权强、风险低或可以无限加杠杆。高信用付款方提高回款稳定性，但高度集中的付款方也可能拥有更强压价能力。

## 清算价值与资产套利专用模块

普通股保守回收价值：

> 现金及证券 + 应收保守回收 + 库存保守回收 + 地产设备变现值 + 子公司/隐藏资产
> − 债务 − 租赁 − 养老金/诉讼 − 优先权/少数股东 − 税费交易成本 − 兑现前现金消耗

必须同时给出：

- 普通股可得回收价值
- 相对市值折价
- 预计兑现时间
- 年化回报
- 管理层或控制股东释放价值的意愿

一票否决包括：价值主要归债权人、持续亏损会吞噬资产、资产无法出售、负债不可验证、管理层无意释放价值。

## 原则坚持与证伪协议

禁止机械使用纯价格止损，但必须区分：

- **价格反对**：股价下跌，核心事实未变。按预案持有、复核或满足条件时加仓。
- **事实反对**：付款方、客户、现金流、资本结构、治理或价值捕获发生恶化。重新估值、减仓或退出。

买入前必须写明：

- 核心事实
- 可容忍价格波动区间
- 重新评估阈值
- 强制退出条件
- 越跌越买条件
- 禁止动作
- 最大等待期限
- 下一验证日期

长期主义不能掩盖事实错误；同时保留资本结构止损、流动性止损、仓位上限和期权到期约束。

## 闭环

机会路径 → 五大支柱 Gate → 研究优先级 → 五维分析 → 估值/Reverse DCF → IDI → 股票或有限风险期权表达 → KPI/Kill Thesis 监控 → 退出复盘。

## 数据纪律

- 同一事实只能归属一个主评分支柱，其他模块只引用，不重复加分。
- 未验证数据不进入评分。
- “大股东买入”“高做空比例”“历史高价”不能单独构成投资理由。
- 所有定性判断必须对应 KPI、阈值、来源和验证日期。
